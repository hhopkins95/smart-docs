# Example Documentation Skill

This is a sample skill for the Smart Docs project that demonstrates how skills can be structured and documented.

## Purpose

This skill helps generate and maintain high-quality documentation by:
- Analyzing code structure
- Generating markdown documentation
- Maintaining consistency across docs

## Usage

The skill is automatically available when working on documentation tasks. It provides context about:
- Markdown best practices
- Documentation structure
- Code-to-doc patterns

## Configuration

No special configuration is required. The skill integrates seamlessly with your workflow.
