# Smart Docs Development Guidelines

This is the project-level context for the Smart Docs project.

## Project Overview

Smart Docs is an AI-native documentation viewer that helps developers visualize and explore:
- Markdown documentation
- Claude Code configuration (skills, commands, agents, hooks)
- Plugin ecosystem
- CLAUDE.md context files

## Code Style

- Use TypeScript for all new files
- Follow existing patterns in the codebase
- Use Tailwind CSS for styling
- Keep components modular and reusable

## Architecture

- **Frontend**: Next.js with React and TypeScript
- **Backend**: Node.js services
- **Real-time**: Socket.IO for file watching
- **Styling**: Tailwind CSS with dark mode support
